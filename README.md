# LiHSI — Listen Help Save International

Static donation site for LiHSI (Listen Help Save International).

## Links

- **Repository:** https://github.com/adjemalkiel/lihsi-donate
- **Live site:** https://adjemalkiel.github.io/lihsi-donate/

## Deployment

The site is published automatically to GitHub Pages on every push to `main` via the workflow in `.github/workflows/deploy-pages.yml`.

If deployment fails with "Get Pages site failed", open **Settings → Pages** on the repository, choose **GitHub Actions** as the source, then re-run the workflow from the **Actions** tab.
